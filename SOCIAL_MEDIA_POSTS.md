Hey everyone! ğŸ‘‹

I just finished building **InterviewQuest** - a gamified CLI tool for practicing coding interviews.

## What is it?

It's a terminal application that turns interview practice into a game. You get code snippets, predict the output, and earn points while managing lives (3 hearts).

## Why?

Let's be honest - interview prep can be boring. I wanted something that:
- Makes practice actually fun
- Lives entirely in the terminal
- Provides instant feedback
- Adapts to your skill level

## Features

- ğŸ¯ **Guess the Output** mode (main game)
- ğŸ’ **Lives system** - lose a heart on wrong answers
- ğŸ”¥ **Streak bonuses** - 3 in a row levels up difficulty
- ğŸ“Š **Score tracking** with accuracy stats
- ğŸ¨ **Beautiful UI** - centered, colorful terminal interface
- ğŸ¤– **AI evaluation** - optional GitHub Copilot integration

## Tech Stack

- TypeScript
- Node.js
- Only 3 dependencies (chalk, figlet, @types/node)
- Custom terminal wrapper
- ~70KB package size

## Try it

```bash
git clone https://github.com/anupamthakur-dev/interview-quest.git
cd interview-quest
npm install && npm run build

# Run with built-in questions
npm start

# Or with AI features
npm run start:ai
```

## Smart AI Configuration

- **Local dev:** AI disabled by default (use `--ai` flag)
- **Production:** AI enabled by default (full experience)
