### With AI Features (Optional)

If users want AI-powered evaluation:

**Linux/Mac:**
```bash
# One-time run
USE_COPILOT=true npm start

# Or if installed globally
USE_COPILOT=true interviewquest
```

**Windows PowerShell:**
```powershell
# One-time run
$env:USE_COPILOT="true"; npm start

# Or if installed globally
$env:USE_COPILOT="true"; interviewquest
```

**Windows CMD:**
```cmd
# One-time run
set USE_COPILOT=true && npm start

# Or if installed globally
set USE_COPILOT=true && interviewquest
```

**Prerequisites for AI:**
- GitHub CLI: `gh --version`
- Copilot extension: `gh extension install github/gh-copilot`
- Authentication: `gh auth login`
